- 라우터 > npm i react-router-dom
- Emotion > npm i @emotion/react
- Recoil > npm i recoil

- Sweetalert2 > mpm i sweetalert2

# JSX 자동완성
- Ctrl + Shift + p rsf?

설정) 선택

```json
    "emmet.syntaxProfiles": {
        "javascript": "jsx"
    },
    "emmet.includeLanguages": {
        "javascript": "html"
    }
```
